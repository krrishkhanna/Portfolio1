# Krrish Khanna Portfolio

Personal portfolio for Founder’s Office, Product, Strategy, and Technical Operations roles. Built with Next.js, TypeScript, Tailwind CSS v4, and deployed on Vercel.

## Live URL

- [portfolio-website-zeta-ten-69.vercel.app](https://portfolio-website-zeta-ten-69.vercel.app)

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Vercel

## Features

- Premium dark portfolio with a refined light mode
- Founder / Operator / Engineer perspective switch
- Structured homepage built around proof, strategy, and execution
- Experience pages framed through ambiguity, structure, and outcomes
- Project case studies with a consistent decision-oriented structure
- Product strategy memo library
- Proof of Work page with curated evidence instead of a certificate dump
- Startup Thinking page for founder-facing essays and startup perspective
- Resume hub with role-specific variants
- Recommendations section with only verified context

## Local Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Validation

```bash
npm run lint
npm run build
```

## Deployment

```bash
npx vercel
npx vercel --prod
```
